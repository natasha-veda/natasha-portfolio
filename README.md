# Natasha Portfolio

Personal portfolio website with cyber-brutalist design aesthetic.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Space Grotesk** - Bold geometric headings
- **JetBrains Mono** - Code snippets

## Design Features

- Cyber-brutalist aesthetic with electric cyan/lime accents
- Animated gradient background blobs
- Scroll-triggered animations
- Custom gradient scrollbar
- Responsive layout
- Subtle noise texture overlay

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Static export ready for deployment to Vercel, Netlify, or any static host.
